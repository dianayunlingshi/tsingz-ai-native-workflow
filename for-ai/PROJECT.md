# PROJECT

## Project

Tsingz AI-Native Workflow Framework

## Purpose

Build professional software for a world where AI is a first-class collaborator.

## Core Idea

Professional software should be designed for humans and AI working together through structured workflows, explicit permissions, shared memory, and plug-and-play components.

## Development Philosophy

Tsingz follows the Tsingz Development Standard (TDS):

- Documentation defines intent.
- Interfaces define contracts.
- Tests define behavior.
- Context guides AI attention.
- Code is implementation.
