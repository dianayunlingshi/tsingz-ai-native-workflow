# START

## Read Order

1. README.md
2. core-principles.md
3. ROADMAP.md
4. for-ai/PROJECT.md
5. for-ai/CURRENT.md
6. The specific `for-ai/*.md` file related to the current task
7. Related docs, interfaces, tests, and source files

## Rule

Do not start from source code alone.

Understand the architecture, interfaces, tests, and current attention focus before generating or modifying implementation.
